Class

Having taken 'Introduction to Designing Online Media,' I gained valuable insights into key concepts and technologies for creating impactful online experiences. This course uniquely combines platform evolution with practical skills in frontend and backend development, offering a comprehensive understanding of online media design.

Class 1: Key Concepts

  • Course Overview and Objectives:
    • Introduction to the course structure, goals, and what students will learn.
  • Definition of Designing and Online Media:
    • Understanding the fundamental concepts of design and online media.
  • Evolution of Online Media Platforms:
    • Tracing the historical development of online media platforms.
  • Key Technologies Driving Platform Development:
    • Exploration of the technologies shaping the online media landscape.
  • Terminology Related to Platform Development:
    • Explanation of key terms used in the context of online media platform development.

Class 2: Platform Architecture and Infrastructure

  • Definitions:
    • Defining key terms related to platform architecture and infrastructure.
  • Client/Server Architecture:
    • Understanding the basic principles of client/server relationships.
  • Client/Server Infrastructure:
    • Examining the infrastructure required for effective client/server interactions.

Class 3: Frontend Development

  • Frontend Development Definitions:
    • Defining key terms related to frontend development.
  • Good UI/UX:
    • Exploring principles of User Interface (UI) and User Experience (UX) design.
  • Key Technologies (Web Development):
    • Overview of essential web development technologies.
  • HTML and CSS:
    • In-depth exploration of HTML and CSS for web development.

Class 4: Frontend Development

  • CSS Layouts:
    • Understanding various CSS layout techniques.
  • JavaScript:
    • Introduction to JavaScript and its role in frontend development.

Class 5: Frontend Development

  • Programming Language Main Concepts:
    • Overview of key concepts in programming languages.
  • JavaScript:
    • Deepening understanding of JavaScript in web development.

Class 6: Backend Development

  • Application Programming Interfaces (APIs):
    • Explanation of APIs and their role in backend development.

Class 7: Backend Development

  • Server, Hosting, and Deployment:
    • Understanding server-related concepts, hosting options, and deployment strategies.
  • Backend Programming:
    • Introduction to backend programming languages and frameworks.
  • Data Storage and Databases:
    • Overview of data storage concepts and databases.

Class 8: Backend Development

  • Command Line Interface (CLI):
    • Introduction to CLI and its role in backend development.
  • Node.js and VSCode:
    • Understanding Node.js, Visual Studio Code, and their application.
  • Dependencies and Application Ports:
    • Discussion on dependencies, application ports, and their significance.
  • Students Backend App Use-Case:
    • Application of knowledge in a practical student-focused backend app use-case.

Class 9: Backend Development

  • Application Debugging:
    • Techniques for debugging backend applications.
  • Students Backend App Use-Case:
    • Continued exploration and troubleshooting of the student-focused backend app.

Class 10: Data Storage and Databases

  • Data Storage:
    • In-depth discussion on various methods of data storage.
  • Databases:
    • Overview of databases and their role in storing and managing data.
  • Types of Databases:
    • Exploration of different types of databases.

Class 11: Full Web Application (Study Case)

  • Version Control System:
    • Introduction to version control, with a focus on GIT/Github.
  • Students App Study Case:
    • Application of knowledge in a comprehensive study case of a student's web application.
  • JavaScript Fetch Method:
    • Understanding and utilizing the JavaScript fetch method in web development.

This course provides a holistic understanding of designing online media, covering both frontend and backend development, along with practical applications and case studies. Further exercises and hands-on practices will be presented on the next page.
